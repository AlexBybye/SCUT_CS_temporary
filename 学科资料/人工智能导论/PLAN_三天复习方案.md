# 三天实干型复习方案（考前冲刺 Plan）

> 考试：今天 2026-07-06，假设 3 天后开考（D-3 → D-1，考试日 D-Day）。
> 题型：20 选择(20分) + 简答(40分) + 计算(40分)。
> 策略：**计算题和简答题是 80 分的主战场，优先级最高**；选择题靠往年卷原题速成。
> 配套资料（本文件夹）：
> - `00_考点分布梳理.md` — 作战地图
> - `01_简答题题库.md` — 简答 + SVM 专题
> - `02_选择题速记与易错点.md` — 选择答案+辨析
> - `03_计算题题型与解法.md` — 6 大计算模板
> - `模拟卷/` — 3 套模拟卷 + 答案

---

## 核心原则（先读）
1. **手推 > 眼看**：计算题一定要拿笔独立推，不能只看答案点头。
2. **模板化**：每类计算题背"步骤模板"，考场套模板。
3. **重点新增必补**：强化学习(MDP)、生成式/判别式、可容性/一致性、SVM——往年卷没有但今年是重点。
4. **每天一套模拟卷**：限时做，对答案，错题回炉。

## 📅 Day 1（D-3）：计算题打地基 + 选择题过一遍

**上午（3h）· 计算题四大件**
- [ ] 读 `03_计算题题型与解法.md` 全文（1h）
- [ ] 手推 **归结证明**：小张快乐、说谎者问题（各1遍）
- [ ] 手推 **贝叶斯网络** P(¬C|¬E)（注意正确答案 0.849，逐项核对）
- [ ] 手推 **决策树 ID3**：记熟 log₂ 常用值，算一道信息增益

**下午（3h）· 计算题另两件 + 神经网络重点**
- [ ] 手推 **α-β 剪枝**：一棵博弈树，标出所有剪枝点
- [ ] 手推 **A\* 八数码**：列前 3 步 OPEN/CLOSED 表
- [ ] **神经网络专题**（重点，多花时间）：
  - AND 神经元不等式组 + 一组解 + 验证；再推 OR、NAND
  - 加法/乘法节点梯度规则；向量计算图 [4,3,5,9] 那题
  - 卷积输出尺寸公式

**晚上（2h）· 选择题速成**
- [ ] 通读 `02_选择题速记与易错点.md`，背标准答案表
- [ ] 重点记"易错辨析"六大块
- [ ] **做模拟卷一的选择题部分**，对答案

✅ Day 1 目标：6 类计算题模板全部手推过一遍；选择题答案背下 70%。

---

## 📅 Day 2（D-2）：简答题攻坚 + SVM + 新增考点 + 全卷模拟

**上午（3h）· 简答题**
- [ ] 通读 `01_简答题题库.md`，逐题盖答案自答
- [ ] **SVM 专题**重点背：优点 + SRM 含义 + U 形图 + 支持向量/核函数（**确定考**）
- [ ] 高频对比题背熟：分类/聚类/回归、符号/连接主义、监督/强化学习、生成式/判别式

**下午（3h）· 新增考点 + 计算题回炉**
- [ ] **强化学习**：MDP 五要素、强化学习目标与基本过程、探索利用（新增必背）
- [ ] 可容性/一致性、推理三层次、归纳/演绎推理
- [ ] 重做 Day1 手推错的计算题
- [ ] **限时做模拟卷二（全卷 2h）**

**晚上（2h）· 模拟卷二订正**
- [ ] 对答案，统计错题，重点回炉错的知识点
- [ ] 神经网络计算题再推一遍（因为是必考复杂版）

✅ Day 2 目标：所有简答能默写要点；SVM 题滚瓜烂熟；强化学习新考点拿下。

---

## 📅 Day 3（D-1）：全真模拟 + 查漏补缺 + 背诵冲刺

**上午（3h）· 全真模拟**
- [ ] **限时做模拟卷三（全卷 2h，严格计时）**
- [ ] 全面对答案，分析失分点

**下午（3h）· 查漏补缺**
- [ ] 把三套模拟卷 + 往年卷所有错题过一遍
- [ ] 计算题：每类再手推 1 遍确保熟练（尤其神经网络、贝叶斯、归结）
- [ ] `00_考点分布梳理.md` 对照，确认无遗漏重点

**晚上（2h）· 背诵冲刺（临门一脚）**
- [ ] 选择题答案表 + 易错点，快速刷一遍
- [ ] 简答要点 + SVM 图，默写
- [ ] 计算题模板"步骤口诀"过一遍：
  - 归结5步、贝叶斯边缘化、α-β(MIN≤α剪/MAX≥β剪)、A*(f=g+h)、ID3(Gain=Ent−条件熵)、梯度(加法直传/乘法交叉)
- [ ] 早睡，带好计算器/文具

✅ Day 3 目标：全卷限时无压力；所有模板/要点能默写；心态稳。

## 🎯 考场时间分配建议（假设 120 分钟）
- 选择题(20分)：**20 min**（会的秒选，不纠结）
- 简答题(40分)：**40 min**（分点作答，写关键词）
- 计算题(40分)：**55 min**（写全过程，先做最有把握的神经网络/决策树/贝叶斯）
- 检查：**5 min**

## ⭐ 必拿分清单（考前最后确认这些都会）
1. ✅ 神经网络：AND 神经元、梯度反传、卷积尺寸
2. ✅ 贝叶斯网络推理（边缘化求条件概率）
3. ✅ 归结证明（5 步法）
4. ✅ 决策树 ID3 信息增益
5. ✅ α-β 剪枝 / A* 表
6. ✅ SVM 简答（SRM + 图）
7. ✅ 强化学习 MDP 五要素
8. ✅ 选择题往年原题答案

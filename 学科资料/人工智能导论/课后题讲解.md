**11. 最短路径子结构性质**

记 d(a,b) 为 a 到 b 的最短代价。已知 s₁→s₂→…→sₖ 是最短路径,由最优子结构:其前缀 s₁→…→sᵢ 是最短的,后缀 sᵢ→…→sₖ 也是最短的,且 d(s₁,sₖ) = d(s₁,sᵢ) + d(sᵢ,sₖ)。 (后缀最短可反证:若存在更短的 sᵢ→…→sₖ,拼接前缀会得到比原路径更短的 s₁→sₖ 路径,矛盾。)

设 P 是任意一条 s₁ 到 sᵢ 的最短路径,cost(P)=d(s₁,sᵢ)。构造 Q = P ⊕ (sᵢ₊₁→…→sₖ),它是合法路径(P 终点为 sᵢ,后段本就从 sᵢ 出发,边都存在),其代价 cost(Q) = cost(P) + cost(sᵢ→…→sₖ) = d(s₁,sᵢ) + d(sᵢ,sₖ) = d(s₁,sₖ)。 即 Q 的代价等于 s₁ 到 sₖ 的最短距离,故 Q 是一条最短路径。∎

**12. MCTS 伪代码**

(1) 因为搜索树中相邻两层对应对弈双方,父结点与子结点的视角相反(零和博弈)。BackPropagate 沿路径回传时每上一层就切换一次视角,所以对当前结点是"得分",对其父结点就是"失分",用**减去**来实现视角翻转。若改为**加上**终局得分,则该得分必须始终以**同一位固定玩家(通常是根结点要走子的那位玩家)的视角**统一计算,回传时不再翻转符号。

(2) 非对抗搜索中没有对手、不存在视角交替,因此:BackPropagate 中把"减去"改为"加上"(始终以同一智能体视角累加);Selection 里 UCB 不再需要按层翻转价值符号(所有层都取最大化);仿真/评估直接用该智能体获得的收益即可。

**13*. alpha-beta 最优情况节点上界 O(b^⌊(m+1)/2⌋)**

按 Knuth–Moore 的结点三分类(理想着法排序):

- 类型1(主变量 PV):所有 b 个孩子都要展开,第 1 个孩子仍是类型1,其余 b−1 个是类型2;
- 类型2(剪枝 CUT):最优情况下第 1 个孩子即产生截断,只展开 1 个孩子,该孩子是类型3;
- 类型3(全展开 ALL):所有 b 个孩子都要展开,每个是类型2。

设深度 h 子树被评估的叶子数,类型1记 f₁(h)、类型2记 f₂(h)、类型3记 f₃(h),f*(0)=1。得递归:

- f₂(h) = f₃(h−1) = b·f₂(h−2), 且 f₂(0)=f₂(1)=1 ⟹ **f₂(h)=b^⌊h/2⌋**
- f₁(h) = f₁(h−1) + (b−1)·f₂(h−1)

逐项求得 f₁(m) = **b^⌈m/2⌉ + b^⌊m/2⌋ − 1**(可验:m=2→2b−1,m=3→b²+b−1,m=4→2b²−1)。

因内部结点数与叶子数同阶,总展开结点数 = O(b^⌈m/2⌉) = O(b^⌊(m+1)/2⌋)(因 ⌈m/2⌉=⌊(m+1)/2⌋)。∎

**14. Dijkstra 与 A***

(1) Dijkstra 就是取 h(n)≡0 的 A*。无负代价时,一致性条件 h(n) ≤ c(n,n') + h(n') 变为 0 ≤ c(n,n'),恒成立;可采纳性 h(n)=0 ≤ h*(n) 也成立。由本章"图搜索 A* 在启发函数一致时最优"的结论,Dijkstra 最优。 (等价的直接论证:Dijkstra 按 g(n) 非降序关闭结点。若某结点 u 关闭时 g(u) 非最短,则最短路上存在尚未关闭、g 值更小的前驱结点先于 u 被关闭,矛盾——这里用到边非负,保证路径代价沿途不减。)

(2) 用一致启发做**重赋权**(Johnson 技巧):令 c'(n,n') = c(n,n') − h(n) + h(n')。 一致性 ⟺ c'(n,n') ≥ 0。对任一路径 s=n₀→…→n_k,重赋权后总代价电话簿式相消: Σc' = (原代价) − h(s) + h(n_k)。 到同一目标的所有路径,c'-代价与原代价只差常数 −h(s)+h(n_k),故 c' 下的最短路 = 原图最短路。又用 A*(启发 h)时 f(n)=g(n)+h(n)=g'(n)+h(s),扩展顺序与在 c' 图上按 g' 的 Dijkstra 完全一致。因 c'≥0,由 (1) Dijkstra 最优,故一致启发下 A* 最优。∎

## 二、简答题
1. 符号主义：智能=符号逻辑运算，基于知识规则推理（专家系统），可解释；连接主义：智能=神经元连接与并行计算，数据训练权重（神经网络），黑盒；行为主义：智能=智能体与环境交互反馈，感知-动作（强化学习、机器人）。

2. 定义：在证据和知识都不完全确定下运用不确定性知识进行的推理。基本问题：①不确定性的表示；②不确定性的传递/传播；③不确定性的合成；④结论更新与决策。

3. 思想：基于密度，用 ε（邻域半径）和 MinPts（最小点数），核心点扩展相连点成簇。过程：①任取未访问点，若为核心点建新簇并纳入其密度可达的所有点；②非核心点标为边界点或噪声；③重复至所有点访问完。缺点：对参数敏感、密度不均效果差、高维距离失效。

4. 生成式学 P(x,y) 再求 P(y|x)、可生成样本（朴素贝叶斯）；判别式直接学 P(y|x) 或决策边界、精度常更高（SVM/逻辑回归/决策树）。

5. 适用（多层）前馈神经网络。过程：前向传播算输出→算损失→误差反向传播用链式法则求各权重梯度→梯度下降更新**相邻层**连接权重和偏置→重复至收敛。

## 三、计算题

**1. α-β 剪枝**
自底向上：
- MIN A 的左 MAX = max(3,5)=5；A 已知 ≤5，更新到 A 时 α（对根 MAX）暂无。
- A 的右 MAX：先看 6，该 MAX ≥6。此时 A 是 MIN，A 当前 = min(5, ≥6)。因右 MAX 值 6 ≥ A 已有的 5？MIN 取小，右 MAX≥6 不会小于 5，故 **A 右 MAX 的第二个叶子 9 被 β 剪枝**（MAX 节点值 6 已 ≥ 上层 MIN 的当前值 5，即 ≥β=5，剪枝）。→ A = 5。
- 根 MAX 得到 A=5，α=5。
- MIN B 的左 MAX = max(1,2)=2。B 是 MIN，B 当前 ≤2。因 2 ≤ 根的 α=5，**B 的右子树整体被 α 剪枝**（MIN 节点值 2 ≤ α=5，剪掉 B 剩余分支，即右 MAX 的 0、7 都不必算）。→ B ≤2。
- 根 = max(5, ≤2) = **5**。
**剪枝位置**：①A 右 MAX 处叶子 9 被剪（β 剪枝）；②B 的整个右 MAX 分支(0,7)被剪（α 剪枝）。根值 = 5。

**2. 决策树 ID3**
总熵：Ent(D)=−(9/14)log₂(9/14)−(5/14)log₂(5/14)
= −(9/14)(−0.637)−(5/14)(−1.485) = 0.4095+0.5304 ≈ **0.940**。
各子集熵：
- Sunny(2正3反)：−(2/5)log₂(2/5)−(3/5)log₂(3/5)=−0.4(−1.322)−0.6(−0.737)=0.529+0.442=**0.971**
- Overcast(4正0反)：熵=**0**（纯）
- Rain(3正2反)：与 Sunny 对称=**0.971**
条件熵：Ent(D,Outlook)=(5/14)(0.971)+(4/14)(0)+(5/14)(0.971)=0.347+0+0.347=**0.694**。
信息增益：Gain(D,Outlook)=0.940−0.694 ≈ **0.246**。

**3. 神经网络前向+反向**
(1) net = 0.5·1 + (−1)·2 + 1 = 0.5−2+1 = **−0.5**。
ŷ = σ(−0.5) = 1/(1+e^{0.5}) ≈ 1/(1+1.6487) ≈ **0.3775**。
(2) ∂L/∂ŷ = ŷ−y = 0.3775−1 = −0.6225。
σ'(net) = ŷ(1−ŷ) = 0.3775·0.6225 ≈ 0.2350。
δ = ∂L/∂net = (ŷ−y)·σ'(net) = −0.6225·0.2350 ≈ **−0.1463**。
- ∂L/∂w₁ = δ·x₁ = −0.1463·1 = **−0.1463**
- ∂L/∂w₂ = δ·x₂ = −0.1463·2 = **−0.2926**
- ∂L/∂b = δ = **−0.1463**

**4. 朴素贝叶斯**
- P(正)·P(f1|正)·P(f2|正) = 0.6·0.8·0.5 = **0.24**
- P(负)·P(f1|负)·P(f2|负) = 0.4·0.3·0.6 = **0.072**
0.24 > 0.072 → 样本 X 属于 **正类**。

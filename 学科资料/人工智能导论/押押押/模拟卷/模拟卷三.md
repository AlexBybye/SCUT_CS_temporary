2. 公式 P(H|E)=P(E|H)P(H)/P(E)。解决逆概率问题：由易得的先验 P(H) 和似然 P(E|H) 求难直接统计的后验 P(H|E)。刻画的是**相关关系**，不是因果关系。

3. k 折交叉验证：数据分 k 份，轮流用 k−1 份训练、1 份验证，取平均性能。用处：充分利用有限数据、可靠评估泛化能力、减小单次划分偶然性、辅助模型选择调参、防过拟合。

4. 可容性：h(n)≤h*(n)，从不高估到目标的真实代价 → 保证 A* 找到最优解。一致性：对相邻 n、n'，h(n)≤c(n,n')+h(n') 且 h(目标)=0 → 一致必可容，且每个节点首次扩展即为最优，无需重开 CLOSED 表节点。

5. 构成：数据采集预处理→特征工程→模型→训练算法→评估验证→部署，评估反馈调参。数据驱动的机器学习是**从数据中归纳规律**（非死记），但结果不完全可预测。风险：数据偏见导致歧视、黑盒不可解释、易被对抗样本攻击、隐私泄露。

## 三、计算题

**1. 说谎者归结**
设 T(x) 表示 x 说真话（老实人），¬T(x) 表示说谎。
- A 说"B、C 都说谎"：若 A 真则 ¬T(B)∧¬T(C)；若 A 假则 ¬(¬T(B)∧¬T(C))=T(B)∨T(C)。
- B 说"A、C 都说谎"：类似。
- C 说"A、B 至少一个说谎"：若 C 真则 ¬T(A)∨¬T(B)。

分析（可用真值表/归结验证）：
- 假设 A 是老实人(T(A))：则 B、C 都说谎(¬T(B),¬T(C))。C 说谎，则 C 的话"¬T(A)∨¬T(B)"为假 → T(A)∧T(B)，但 T(B) 与 ¬T(B) 矛盾。故 A 不是老实人。
- 假设 B 是老实人(T(B))：则 A、C 都说谎。C 说谎则 T(A)∧T(B)，与 A 说谎(¬T(A))矛盾。故 B 不是老实人。
- 故 A、B 都说谎。C 说的"A、B 至少一个说谎"为真 → **C 说真话，C 是老实人**。且题设"有人从不说真话"由 A、B 满足。
**结论：C 是老实人，A、B 是说谎者。**

**2. A\* 搜索**
f=g+h。
- 初始：OPEN=[s(f=0+6=6)]，扩展 s。
- 扩展 s：生成 A(g=1,f=1+4=5)、B(g=4,f=4+2=6)。OPEN=[A(5), B(6)]，CLOSED={s}。取 A(f=5)扩展。
- 扩展 A：生成 C(g=1+2=3,f=3+2=5)、G(g=1+12=13,f=13+0=13)。OPEN=[C(5), B(6), G(13)]，CLOSED={s,A}。取 C(f=5)扩展。
  （注：经 B 到 C 的 g=4+1=5>3，故 C 保留经 A 的 g=3）
- 扩展 C：生成 G(g=3+3=6,f=6+0=6)，更新 G（6<13）。OPEN=[G(6), B(6)]，CLOSED={s,A,C}。取 G(f=6)扩展 → 到达目标。
**最优路径：s→A→C→G，总代价 g=6。**
OPEN 表变化：[s:6]→[A:5,B:6]→[C:5,B:6,G:13]→[G:6,B:6]→目标。

**4. 卷积 + 梯度**
(1) 输出 =(输入−核+2·填充)/步幅+1 =(5−3+2·1)/1+1 =(4)/1+1 = **5×5**。
(2) 乘法节点梯度**交叉相乘**：∂z/∂x=y=4，∂z/∂y=x=3。
- x 处梯度 = 上游梯度·y = 5·4 = **20**
- y 处梯度 = 上游梯度·x = 5·3 = **15**

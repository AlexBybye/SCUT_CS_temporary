## 三、计算题

**1. 归结证明"某些整数是偶数"**
谓词：N(x) 自然数，I(x) 整数，E(x) 偶数。
- 前提1 ∀x(N(x)→I(x)) → 子句 C1: ¬N(x)∨I(x)
- 前提2 ∃x(N(x)∧E(x)) → Skolem 化，设常量 a：C2: N(a)，C3: E(a)
- 结论 ∃x(I(x)∧E(x)) 取否：¬∃x(I(x)∧E(x)) = ∀x(¬I(x)∨¬E(x)) → C4: ¬I(x)∨¬E(x)

归结：
1. C1{x/a} + C2 ⟹ I(a)
2. C4{x/a} + I(a) ⟹ ¬E(a)
3. ② + C3 ⟹ **□**
∴ 某些整数是偶数，得证。

**2. P(¬C|¬E)**
P(¬E|·)=1−P(E|·)：P(¬E|S,C)=0.1, P(¬E|S,¬C)=0.7, P(¬E|¬S,C)=0.5, P(¬E|¬S,¬C)=0.9。
- P(¬E)=0.1·0.4·0.3 + 0.7·0.4·0.7 + 0.5·0.6·0.3 + 0.9·0.6·0.7
      = 0.012+0.196+0.090+0.378 = **0.676**
- P(¬C,¬E)=0.7·0.4·0.7 + 0.9·0.6·0.7 = 0.196+0.378 = **0.574**
- P(¬C|¬E)=0.574/0.676 ≈ **0.849**

**3.（10 分）神经元实现 AND 函数。**

**3. AND 神经元**
约束（输出为 0 需 net<0，输出 1 需 net≥0）：
```
(0,0): b < 0
(0,1): w₂+b < 0
(1,0): w₁+b < 0
(1,1): w₁+w₂+b ≥ 0
```
取 **w₁=1, w₂=1, b=−1.5**。验证：0+0−1.5=−1.5<0→0✓；1−1.5=−0.5<0→0✓；1−1.5<0→0✓；2−1.5=0.5≥0→1✓。（解不唯一）
## 题4变式 · 计算图梯度反传(含乘法节点)

**题目**

某运算得到向量

```
y = [ x1·x2,  x2 + x3,  x3·x4,  x4 ]
```

已知输入值 x = [x1, x2, x3, x4] = [2, 3, 4, 5],损失 L 对 y 的梯度为

```
∂L/∂y = [4, -1, 5, 9]
```

求 L 对 x = [x1, x2, x3, x4] 的梯度。写出计算过程。

---

**分析:先看每个 x 影响了哪些 y**

```
y1 = x1·x2      ← 乘法，含 x1, x2
y2 = x2 + x3    ← 加法，含 x2, x3
y3 = x3·x4      ← 乘法，含 x3, x4
y4 = x4         ← 直传，含 x4
```

所以:x1 只进 y1;x2 进 y1、y2;x3 进 y2、y3;x4 进 y3、y4。凡是进了多个 y 的,梯度要**相加**。

**逐个求梯度(乘法交叉、加法直传)**

```
∂L/∂x1 = ∂L/∂y1 · ∂y1/∂x1
       = 4 · x2                （乘法交叉：对 x1 求导得另一路 x2）
       = 4 · 3 = 12

∂L/∂x2 = ∂L/∂y1 · ∂y1/∂x2  +  ∂L/∂y2 · ∂y2/∂x2
       = 4 · x1        +  (-1) · 1     （y1 乘法交叉得 x1；y2 加法直传得 1）
       = 4·2 + (-1)
       = 8 - 1 = 7

∂L/∂x3 = ∂L/∂y2 · ∂y2/∂x3  +  ∂L/∂y3 · ∂y3/∂x3
       = (-1) · 1       +  5 · x4      （y2 加法直传；y3 乘法交叉得 x4）
       = -1 + 5·5
       = -1 + 25 = 24

∂L/∂x4 = ∂L/∂y3 · ∂y3/∂x4  +  ∂L/∂y4 · ∂y4/∂x4
       = 5 · x3         +  9 · 1       （y3 乘法交叉得 x3；y4 直传）
       = 5·4 + 9
       = 20 + 9 = 29
```

**答案**

```
∂L/∂x = [12, 7, 24, 29]
```
